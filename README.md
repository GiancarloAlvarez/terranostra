# Terra Nostra · Landing de Restaurante

Plantilla landing page para restaurantes, construida con HTML5, CSS3 y JavaScript vanilla.

## 🚀 Características
- Diseño responsive (mobile-first)
- Sistema de colores personalizable con variables CSS
- Código limpio y comentado

## 🎨 Personalización
Edita las variables en `css/styles.css` dentro de `:root` para cambiar colores y fuentes.

## 📦 Instalación
1. Descarga o clona el repositorio.
2. Abre `index.html` en tu navegador (o usa Live Server).
